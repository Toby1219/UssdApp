UssdApp
=======

Description
-----------
UssdApp is a USSD (Unstructured Supplementary Service Data) application project. This repository contains the source code and resources for building and running the USSD-based service/app.

Features
--------
- Basic USSD session handling
- Menu-driven user interactions
- Simple backend logic (check code for specifics)

Getting Started
---------------
1. Clone the repo:
   git clone https://github.com/Toby1219/UssdApp.git

2. Open the project in your preferred IDE.

3. To run use:
   - cargo run (to run app)
   - cargo test (to test app)
            or 
      cargo test -- --show-output (to show stdout message)
